================
レシピブックとは
================

「レシピブック」はPythonで書かれたレシピサイト向けCMSです。

Djangoフレームワークの上に構築されており、以下の特徴があります。

* レシピコンテストの開催に対応
* Ajaxを活用したユーザーインターフェース
* Django標準の使いやすい管理画面

このドキュメントを見ながら、「レシピブック」を試してみましょう。
